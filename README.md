# Komerco is a commerce based on Angular 4+, Bootstrap 4 and <a href="https://github.com/akveo/nebular">Nebular</a>

### What's included:

- Angular 4+ & Typescript
- Bootstrap 4+ & SCSS
- Responsive layout
- High resolution
- Authentication module with multiple providers
  
And many more!
